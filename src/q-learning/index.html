Q-Learning Visualizer

Usage

  • 青いのはスタート、赤いのはゴール、黒いのは敵です。
  • 矢印の濃さが Q 値を表します。 白に近いほど 0 に近く、緑に近いほど 100 に近いです。
  • s00 a0, s00 a1, s00 a2, s00 a3, s01 a0, ..., s04 a3, s10 a0, ... の順番に空白区切りで入力してください。
  • スマホ利用は想定していません。 見づらいかもしれませんが、動くとは思います。
