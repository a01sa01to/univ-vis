Q-Learning Visualizer

Usage

  • 青いのはスタート、赤いのはゴール、黒いのは敵です。
  • 矢印の濃さが Q 値を表します。 白に近いほど 0 に近く、緑に近いほど 100 に近いです。
  • 矢印の上にカーソルを合わせると、 Q 値が表示されます。
  • Q 値は、0.0 から 100.0 までの値を取ります。
  • s00 a0, s00 a1, s00 a2, s00 a3, s01 a0, ..., s04 a3, s10 a0, ... の順番に空白区切りで 1 行ずつ入力してください。
  • スマホ利用は想定していません。 見づらいかもしれませんが、動くとは思います。
